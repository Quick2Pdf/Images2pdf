# Image to PDF Converter (Web App)

A free and easy-to-use web tool to convert images into PDF with the following features:

- 📤 Upload multiple images
- ✂️ Auto crop & resize
- 💡 Brightness filter
- 🌓 Dark mode toggle
- 📥 One-click PDF download
- 🖱 Drag & Drop support
